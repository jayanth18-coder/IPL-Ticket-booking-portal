# IPLTickets2022

IPLTickets2022 is a Java-based web application developed using JSP, Servlets, HTML, CSS, and JavaScript for online IPL ticket booking and management.

## Features

- User Registration
- User Login
- IPL Match Schedule
- Ticket Booking
- Booking History
- Admin Panel
- JSP and Servlet Based Architecture

## Technologies Used

- Java
- JSP
- Servlets
- HTML
- CSS
- JavaScript
- MySQL (Optional)

## Project Structure

- `web/` - JSP pages, CSS, images
- `src/` - Java source files
- `WEB-INF/` - Configuration files

## Note

This project was developed as an academic project for learning Java Web Application Development.
