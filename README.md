# IPL Ticket Booking Portal

A Java-based web application developed using JSP, Servlets, HTML, CSS, and JavaScript for managing IPL match ticket bookings online. This project was created as an academic web development project to demonstrate ticket reservation, user authentication, and administrative management functionalities.

## Features

### User Module

* User Registration
* User Login
* View IPL Match Schedule
* Book Match Tickets
* View Booking History
* Forgot Password Functionality

### Admin Module

* Admin Login
* Add Match Details
* Manage Match Schedules
* View User Bookings
* Manage Ticket Availability

## Technologies Used

* Java
* JSP (Java Server Pages)
* Servlets
* HTML5
* CSS3
* JavaScript
* Apache Tomcat
* MySQL Database
* JDBC

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/jayanth18-coder/IPL-Ticket-booking-portal.git
```

2. Open the project in NetBeans or VS Code.

3. Configure Apache Tomcat Server.

4. Configure MySQL database and update database connection settings.

5. Deploy the project and run it on Tomcat.
